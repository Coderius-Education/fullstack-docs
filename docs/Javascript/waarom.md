# Waarom Javascript?
Tot nu toe hebben de webpagina's die je gemaakt hebt met HTML en CSS er voor iedereen vrijwel altijd hetzelfde uitgezien.
Gelukkig heb je al gezien dat met CSS pseudoclass je al iets kon laten veranderen door de gebruiker.
Met Javascript kun je nog veel meer!

Laten we eerst het programma gaan downloaden om code in te schrijven.
