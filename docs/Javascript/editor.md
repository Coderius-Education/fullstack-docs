# Visual Studio Code

## Stap 1: Download Visual Studio Code
Via deze link https://code.visualstudio.com/ kun je het programma downloaden en installeren.

## Stap 2: een folder kiezen
Open het programma Visual Studio Code. Je ziet waarschijnlijk iets dat lijkt op:

![start](@site/static/vs_code/start.png)

Klik op `Open Folder` en kies een folder op je computer waarin je de bestanden van je website gaat opslaan.

:::warning Pas op
Zorg dat je deze folder kan terugvinden als je op een later moment verder wilt met je project.
:::

## Stap 3: maak index.html
Klik met je rechtermuisknop in de linkerkant van Visual Studio Code en kies `New File`

![start](@site/static/vs_code/new_file.png)

## Stap 4: hello world
Typ in `index.html` wat HTML, bijvoorbeeld:

![start](@site/static/vs_code/hello_world.png)

## Stap 5: opslaan
Kies in het Menu `File` en klik dan op `Auto Save`, dan worden al je wijzigingen altijd meteen opgeslagen.

## Stap 6: Website tonen
- Klik met je rechtermuisknop op `index.html`
- Kies `Reveal in File Explorer`
- Als het goed is opent `Windows Verkenner` met daarin het bestand `index.html`
- dubbelklik op het bestand `index.html` en je website zou moeten openen in een browser.
