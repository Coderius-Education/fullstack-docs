De volgende lessen vinden wij erg nuttig:
- Les 1 - Hello world
- Les 2 - Elementen uit je webpagina herhalingen
- Les 3 - Objecten, puntnotatie en style
- Les 4 - Fouten opsporen met de console
- Les 5 - Invoer - Klikken op een knop
