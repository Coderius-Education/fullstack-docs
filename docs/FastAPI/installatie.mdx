# Installatie

Tijd voor een back-end!
We gaan er wel vanuit dat je:
- een cursus Python hebt gevolgd
- Visual Studio Code geïnstalleerd hebt op je computer
- Python geïnstalleerd hebt op je computer
- een virtual environment hebt aangemaakt in je project in Visual Studio Code.

We gaan er dus vanuit je Visual Studio Code er ongeveer zo uitziet.

Let op: het is essentieel dat in je terminal het pad begint met `(.venv)`:

:::warning Nog niet alles geïnstalleerd?
Ga dan naar [hier](https://play.coderius.nl/docs/installatie/VScode) om alles geïnstalleerd te krijgen.
:::

![venv](@site/static/vs_code/venv.png)

## FastAPI installeren
Kopieer het volgende commando naar de terminal in Visual Studio Code en druk op Enter:

```bash
pip install "fastapi[standard]"
```
